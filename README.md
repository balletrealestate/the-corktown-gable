# The Corktown Gable 🏠

**An Open-Source Home Design for Detroit.**

Built in the historic North Corktown neighborhood, this repository contains the complete, open-source architectural plans for a modern, city-approved single-family home, including a detached garage and a carriage house apartment.

By releasing this intellectual property into the public domain via a Creative Commons license, Ballet Real Estate aims to support [Mayor Sheffield and BSEED’s initiative](https://detroitmi.gov/news/mayor-bseed-director-announce-sweeping-plan-fast-track-renovations-existing-homes-construction-new) to build 1,000 new single-family homes in Detroit by eliminating upfront architectural costs and fast-tracking the construction process.

---

## 📖 Open Source Architecture: What Does This Mean?

Traditionally, if you want to build a house, you have to pay tens of thousands of dollars to an architect to draw up plans from scratch, and then spend months getting those custom plans approved by the city. 

We are treating this house like open-source software. This repository is licensed under the **[Creative Commons Attribution 4.0 International (CC BY 4.0) License](LICENSE.md)**. 

**In simple terms, this means you are absolutely free to:**
*   **Share:** Copy and redistribute the blueprints in any medium or format.
*   **Adapt:** Remix, alter, and build upon the plans (e.g., change the siding, move a window, or update the electrical).
*   **Build & Sell:** You can legally build this house for yourself, or build it commercially to sell or rent.

**The only requirement:** You must give appropriate credit to Ballet Real Estate and provide a link to the license. That's it. No royalties, no red tape.

## 🗂️ Repository Structure

This repository contains both the "ready to print" files for the city, and the "raw source code" for architects and engineers who want to make modifications.

| Directory | Contents | Target Audience |
| :--- | :--- | :--- |
| `/pdfs` | Fully rendered, printable architectural blueprints, elevations, and site plans. | Homeowners, Builders, BSEED Permitting |
| `/raw-files` | Editable CAD/BIM files (e.g., .dwg, .skp, .rvt) to modify the structural or cosmetic design. | Architects, Engineers, Designers |
| `/assets` | High-resolution project photos and pre-construction renderings. | Media, Neighborhood Associations |
| `/specs` | (Coming Soon) The Bill of Materials (BOM), energy efficiency specs, and HVAC sizing. | General Contractors, Estimators |

## 🚀 "Deploy to Detroit" Guide

We built this specifically to align with the City of Detroit's push for **Pre-approved New Home Designs**. While BSEED is finalizing that official program, here is how you can use these plans today:

1. **Download the PDF Plans:** Grab the latest release from the `/pdfs` folder.
2. **Site Adaptation:** You will need a local surveyor/engineer to create a site plan mapping this house to your specific Detroit lot.
3. **Submit to BSEED:** Because this exact structure has already been permitted and built in Corktown, BSEED reviewers have a precedent for approving its structural integrity and code compliance. 
4. **Build:** Use the upcoming Bill of Materials in our `/specs` folder to get accurate bids from Detroit-based general contractors.
